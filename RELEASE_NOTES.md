## Релиз версии [3.0.2] от 29.05.2026

### Новое
- **Добавлен эндпоинт `Добавлен endpoint /webm2mp4`:** — конвертация WEBM → MP4 (H264 + AAC) с поддержкой браузеров, Android, Telegram, WebView и MediaRecorder API.
- **Добавлен эндпоинт `Добавлен endpoint /webp2png`:** — конвертация WEBP → PNG с сохранением прозрачности.
- **Добавлен эндпоинт `Добавлен endpoint /heic2jpg`:** — конвертация HEIC/HEIF (iPhone/iOS) → JPG.

## Релиз версии [3.0.1] от 06.04.2026

### Новое
- **Добавлен эндпоинт `/wav2mp3`:** Теперь поддерживается конвертация аудиофайлов из формата WAV в MP3. Это расширяет возможности сервиса по обработке различных аудиоформатов.

## Релиз версии [3.0.0] от 23.03.2026

### Изменения
- **Переезд на Yandex.Cloud:** Проект полностью перенесен с Render на Yandex.Cloud Serverless Functions.
- **Новый URL:** Сервис теперь доступен по адресу [https://d5dtt5rfr7nk66bbrec2.kf69zffa.apigw.yandexcloud.net/converter](https://d5dtt5rfr7nk66bbrec2.kf69zffa.apigw.yandexcloud.net/converter).
- **Обновлена документация:** README.md обновлен для отражения изменений в инфраструктуре и эндпоинтах.
- **Архитектурное изменение:** Проект полностью переведен с модели Yandex.Cloud Functions на запуск в Docker-контейнере. Это обеспечивает большую гибкость, предсказуемость окружения и упрощает локальную разработку.
- Удалена зависимость `serverless-http`, которая была необходима для запуска в формате облачной функции.
- Express-сервер теперь запускается напрямую, прослушивая порт, указанный в переменной окружения `PORT` (или 8080 по умолчанию).
